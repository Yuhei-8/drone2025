**ドローンジャーナリズムと戦争兵器としてのドローン最前線

— 大田区の空を見上げながら考えた「空撮の未来と危うさ」**

羽田空港がすぐそばにある大田区に住んでいると、空を見上げる機会が自然と増える。飛行機だけでなく、最近ではドローンもよく見かけるようになった。昔センターを守っていた頃、飛球（フライ）の軌道を読むのが得意だったけれど、その感覚で空を見ていると、ドローンは“新しいボール”のように思える時がある。静かに、しかし確実に社会のど真ん中へ飛び込んできている。

■ ドローンジャーナリズム：空から「真実」に近づく新しい目

まずは平和利用の側面から。
ドローンは近年、ジャーナリズムの領域で急速に存在感を高めている。理由は明確で、危険地帯・立ち入り禁止区域・災害現場などを安全かつ迅速に撮影できるからだ。

土砂崩れ現場での上空撮影

洪水被害の広域観測

山火事の規模把握

建物倒壊地域の捜索支援

こうした場面では、ドローンが“記者の代わりに”現場に入り、被害の全容を伝える。
これは、すし屋で握りながらニュースを見ていても感じるが、映像の迫力だけでなく、情報の正確性という点で大きな意味がある。

さらに、海外紛争地域ではドローンによる空撮映像が、政府や軍が公開する公式発表とは異なる“地上のリアル”を届けるケースも増えている。
言わば、空からのジャーナリズムは、権力と市民の間に新しい透明性をつくるツールとして育っている。

昔センターで打球を追っていた自分からすると、ドローンは“高いところから全体を読む力”を人間に与えてくれる存在だと感じる。

■ 一方で急速に進む軍事利用：戦争のルールを変える兵器へ

しかし同じドローンが、戦争ではまったく別の顔を見せる。
ここ数年、軍事ドローンはウクライナ戦争をはじめ、多くの紛争で“戦争の主役”になりつつある。

● 特徴①：安価で扱いやすく、数で押せる

高価な戦闘機とは違い、改造された市販ドローンでも十分な戦術効果を持つ。
まるで「守備の反応が速い選手をたくさんベンチに置く」ように、数を揃えることで圧倒的な優位を生む。

● 特徴②：無人だからリスクが低い

操縦者は遠隔にいるため、死傷リスクはほぼゼロ。
その結果、攻撃のハードルが下がり、戦争が“軽く始まる”危険性が指摘される。

● 特徴③：精密攻撃と監視が容易

高性能カメラ・赤外線センサー・AI自動追尾などの技術により、
「どこに何があるか」を詳細に把握しながらの攻撃が可能になった。

しかし、これらの進化は同時に民間地帯の攻撃や誤爆リスクを高めている。
空を飛ぶものがすべて武器かもしれないという緊張感は、地上の人々の生活をむしばむ。

■ 空をどう使うかは人間次第

筋トレのように、技術は鍛えれば鍛えるほど強力になる。
それが正しく使われるか、危険な方向に使われるかは、結局人次第だ。

ドローンは、

災害現場を救う“目”にも、

戦場を激化させる“刃”にもなる。

センターとして外野の広い視野を大事にしてきた自分としては、
「上から見えるからこそ守れるもの」と
「上から見えるからこそ壊してしまうもの」
その両方を意識する必要があると感じる。

■ まとめ：大田区の空から考えるドローンの未来

羽田空港の近くで、飛行機とともにドローンを見上げる生活。
バイクに乗って風を切る瞬間に、空を滑る小さな機体を見ると、
時代が変わっていく速さを肌で感じる。

ドローンは、
社会を映す鏡であり、未来を映すレンズでもある。

それが“救う”方向に飛んでいくのか、
あるいは“傷つける”方向に使われてしまうのか。
その境界線は、今この瞬間の私たちの選択にかかっている。
ドローンジャーナリズム
